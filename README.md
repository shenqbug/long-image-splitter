✂️长图分割器 (Long Image Splitter)
---
一个纯前端的长图切分工具，可将长截图拆分为多张子图，并提供“智能切分”策略，尽量减少在文本区域中间截断的情况。

在线使用：https://shenqbug.github.io/long-image-splitter/


动机
---
有些时候，我们需要将一些很长的截图上传给ai助手，但是长宽比例过于大的图片又会导致OCR失效或者结果乱码（经测试，至少有三个主流ai助手如此）。 

为解决这个问题，做了一个小网页，支持长图片自动分割（分割线远离文本），支持打包下载或复制。



功能
---
- 支持本地图片上传（拖拽 / 文件选择）
- 提供两种切分模式：
  - 智能切分：在目标位置附近搜索较平滑区域作为切点
  - 固定切分：按设定高度等距裁剪（支持重叠）
- 提供切分区域的实时预览
- 支持单张图片保存
- 支持批量下载（ZIP 打包）
- 支持复制单张图片及拼接后的整图（受浏览器限制）
- 所有处理均在浏览器本地完成，不上传数据


原理
---

### 图像处理流程

本地文件 → FileReader → Image → Canvas → 切分 → 导出


- 使用 `FileReader` 读取用户选择的图片
- 通过 `Canvas` 完成图像裁剪与绘制
- 每个子图对应一个独立的 canvas


### 智能切分策略

在固定切分高度的基础上，对切分位置进行局部搜索：

1. 设定理论切点（start + chunkHeight）
2. 在上下小范围内扫描候选行
3. 对每一行计算“像素变化程度”（灰度差 + 深色像素比例）
4. 选择变化较小的行作为最终切点

基本假设：

- 文本区域：像素变化频繁
- 空白/背景区域：像素变化较小

该方法属于启发式策略，计算开销低，适合前端实时处理场景。
